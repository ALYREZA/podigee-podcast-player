We are testing a new podcast player.
1x

This is a work in progress. It is lacking organization of code as well as some essential features.

We are developing the player to be used on our Podcast Publishing Platform Podigee. Use in other areas is highly encouraged though!

Technically it is based on the excellent HTML5 Media Element API and mediaelement.js (e.g. for flash fallback).

Find a list of current as well as future features below. If you have ideas or wishes regarding features or capabilities, or would like to contribute, please get in touch:

hello@podigee.com

Current capabilities

  • HTML5 first
  • play/pause, fast-forward/-backward
  • playback speed
  • drag to seek
  • click to jump
  • loading animation
  • show time played/time left
  • display logo/title/description

Capabilities to come

  • flash fallback
  • cross browser compatibility
  • browser feature detection
  • chapter mark list
  • jump to timestamp from URL
  • embedding
  • playlist support
  • fully configurable/themable
  • plugin support
